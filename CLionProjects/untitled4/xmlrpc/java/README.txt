The Apache XML-RPC library is here: https://ws.apache.org/xmlrpc/
Note that this is Version 2.0.1 of the library.

You must first install the Java JRE and SDK.

sudo apt-get update
sudo apt-get install default-jre
sudo apt-get install default-jdk

The archives, including javadocs, can be found here: https://archive.apache.org/dist/ws/xmlrpc/binaries/

To compile type:
	make all

To run the server type:
	make server

To run the client type:
	make client
