The node.js library is from here: https://github.com/baalexander/node-xmlrpc

To use the node.js example, you need to install npm, the xmlrpc module, and nodejs.
To do this, run the following commands:

sudo apt-get update
sudo apt-get install npm
npm install xmlrpc
sudo apt-get install node

To run the server type:

make server

Then to run the client type:

make client
