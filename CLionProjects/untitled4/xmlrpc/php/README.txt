The php XML-RPC library here is from http://scripts.incutio.com/xmlrpc/

To use this example, you must first install php as below.

sudo apt-get update
sudo apt-get install php

To run the server type:

make server

To run the client, open another terminal window and type:

make client

To execute the client, you must open Firefox and go to web address below:

http://localhost:8000/client.php

The client.php file will execute the server.php code at port 8080
